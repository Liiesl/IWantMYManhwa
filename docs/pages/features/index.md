# Features

The Manhwa Downloader extension provides powerful features for downloading and organizing your favorite manhwa and manhua chapters.

## Chapter Scanning

Automatically detects all available chapters on a series page. No manual copy-pasting of URLs needed.

## Range Selection

Download specific chapter ranges instead of everything:
- Select start and end chapters
- Download only the chapters you need
- Perfect for catching up on recent releases

## Progress Tracking

Monitor download progress in real-time:
- Individual chapter progress bars
- Status updates for each chapter
- Overall download progress

## Pause and Resume

Control your downloads:
- **Pause**: Stop new chapters from starting
- **Resume**: Continue downloading from where you left off
- **Stop**: Cancel all remaining downloads

## Multi-Site Support

Works with various manhwa and manhua sites through an adapter system. New sites can be added by creating adapters.

## Organized Output

Downloads are organized automatically:
- Each chapter in its own folder
- Images numbered sequentially
- Series title included in folder structure
- Everything packaged in a single ZIP file

## ZIP Compression

All chapters are compressed into a single ZIP file for easy:
- Storage
- Transfer
- Organization
