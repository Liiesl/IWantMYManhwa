# Getting Started

Welcome to the Manhwa Downloader documentation. This guide will help you understand how to use the extension effectively.

## Quick Start

1. Navigate to your favorite manhwa/manhua website
2. Click the **Scan** button in the extension panel
3. Select the chapters you want to download
4. Click **Download** to start

## Installation

The extension is loaded as an unpacked extension in Chrome developer mode:

1. Download the extension files
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked** and select the extension folder

## First Use

When you first open the extension on a supported site:

- The extension will detect the site automatically
- Click **Scan** to find available chapters
- The series title and chapter count will appear
- Use the range inputs to select which chapters to download

## Tips for Best Results

- Make sure you're on a series page (not a chapter page) when scanning
- Wait for the page to fully load before clicking Scan
- The extension works best on supported sites listed in the documentation
