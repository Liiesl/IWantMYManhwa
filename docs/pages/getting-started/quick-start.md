# Quick Start Guide

This guide will get you downloading chapters in under 2 minutes.

## Step 1: Navigate to a Series Page

Go to any supported manhwa site and open a series page. For example:
- `https://webtoonscan.com/manhwa/solo-leveling`
- `https://asuracomic.net/series/solo-leveling`

## Step 2: Open the Extension

Click the extension icon in your browser toolbar or use the side panel.

## Step 3: Scan for Chapters

Click the **Scan** button. The extension will:
- Detect the series title
- Find all available chapters
- Display the total count

## Step 4: Select Chapter Range

Use the Start and End inputs to choose which chapters to download:
- **Start**: First chapter to download
- **End**: Last chapter to download

## Step 5: Download

Click the **Download** button. The extension will:
- Download each chapter's images
- Create a ZIP file organized by chapter
- Save it to your Downloads folder

## During Download

You can:
- **Pause**: Temporarily stop downloading
- **Resume**: Continue from where you paused  
- **Stop**: Cancel remaining downloads

## Accessing Your Files

Downloaded ZIP files are saved to your browser's default download location. Extract them to read offline!
