Manhwa Downloader Documentation

Getting Started

Welcome to the Manhwa Downloader documentation. This guide will help you understand how to use the extension effectively.

Quick Start

  1. Navigate to your favorite manhwa/manhua website
  2. Click the "Scan" button in the extension panel
  3. Select the chapters you want to download
  4. Click "Download" to start

Features

  • Chapter Scanning - Automatically detects available chapters
  • Range Selection - Download specific chapter ranges
  • Progress Tracking - Monitor download progress in real-time
  • Pause/Resume - Control downloads with pause and stop buttons
